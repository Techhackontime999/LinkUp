HOW TO USE
================================================================================

1. Start Django:
   cd linkup
   python manage.py runserver

2. Start Redis (for WebSocket):
   redis-server

3. Access Admin:
   http://localhost:8000/api/admin/

4. Register Agent:
   - Web UI: Click "Add New Model"
   - API: POST /api/agents/register

5. Authenticate:
   POST /api/agents/authenticate

6. Send Message:
   POST /api/messages

7. WebSocket:
   ws://localhost:8000/ws/agent/{agent_id}/?token={jwt}
